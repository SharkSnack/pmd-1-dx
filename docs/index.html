This is a wiki for Pokemon Mystery Dungeon: Rescue Team DX for the Nintendo Switch.
It's a work-in-progress, so some pages don't have content yet, and others may change in time.

If you spot any errors or wish to contribute, please join the community Discord server and ping shark.

A significant portion of the information found on this site is hand-translated from the Japanese wiki,
along with information others found through datamining + some information from Serebii and Bulbapedia.
The goal of this site is to organize things into one spot so information is easier to access.
