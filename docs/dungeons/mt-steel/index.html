Mt. Steel

Edit on GitHub

Dungeon that slightly dials up the difficulty with a simple boss fight at the end.
The party's HP, PP, and Belly is fully restored before the boss, so there's no need to heal up beforehand.
You can be rescued by another player in this game even if you wipe out during a boss fight,
so there isn't a whole lot of danger of losing items unless you choose not to use rescues.

If you return to this dungeon after clearing it, you might run into a Rapidash Strong Foe.
Strong Foes are powerful pokemon that you should generally run away from at this point in the game.
They always drop a Deluxe Box and offer a lot of experience points if you manage to defeat them,
but they can't be recruited unless you have a Friend Bow (obtained in a post-game dungeon).
