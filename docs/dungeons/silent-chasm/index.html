Silent Chasm

Edit on GitHub

This time there isn't a boss fight, so the goal is to simply reach the target depth.
Weedle can eat food and seed items (including Tiny Reviver Seeds) in your Toolbox using Bug Bite.
Teddiursa, Farfetch'd, and Beedrill can have powerful multi-hit attacks that potentially one-shot you.
Beedrill in particular can use Twineedle from up to 4 tiles away, so beware if you're a Grass type.
