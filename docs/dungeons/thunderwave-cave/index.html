Thunderwave

Edit on GitHub

Dungeon that notably introduces some electric type Pokemon who can cause Paralyzed status.
Unlike the DS version, most Electric types here can use Electric type attacks like Thunder Shock or Spark,
so you might want to be a little careful around them if you chose a Water type starter or partner.
Otherwise, the only thing to be careful of is Nidoran's Poison Point ability.
