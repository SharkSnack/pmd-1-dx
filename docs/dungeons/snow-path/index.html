Snow Path

Edit on GitHub

Optional dungeon you can challenge before you enter Frosty Forest or Mt. Freeze.
You just return to the location before you entered Snow Path after clearing it, and there's no reward.
Enemies offer more experience points compared to Rock Path, but that's about all there is to say about it.
