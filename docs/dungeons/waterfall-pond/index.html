Waterfall Pond

Edit on GitHub

Optional post-game dungeon featuring many Water types that don't appear in sea type dungeons.
Gyarados (Strong Foe) often causes accidents thanks to its high stats + Hydro Pump or Hyper Beam.
Food items (including seeds and berries) are shop-exclusive and can't be found on the ground.
Evolution Crystals (1000 Poke) can be found in shops, so don't forget to bring some money.

Enemies with Swift Swim can overlap with some Rain Dance users / rainy weather floors:

  • Swift Swim: Lotad (1-3F), Goldeen (1-5F), Surskit (4-7F), Magikarp (7-11F), Lombre (13-16F), Feebas (16-18F)
  • Rain Dance: Lotad (1-3F), Poliwag (1-5F), Gyarados (5-18F), Poliwhirl (9-13F)
