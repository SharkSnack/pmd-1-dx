Rock Path

Edit on GitHub

Optional dungeon you can challenge only if you wipe out in Lapis Cave or before you enter Mt. Blaze.
There's a very high chance for a shop to generate on B4F, so the idea is to restock items here if needed.
You just return to the location before you entered Rock Path after clearing it, and there's no reward.
