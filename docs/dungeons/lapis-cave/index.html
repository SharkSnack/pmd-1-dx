Lapis Cave

Edit on GitHub

First dungeon of the fugitive arc, and the player's visibility in hallways is limited to a 2 tile radius.
However, story dungeons always have Radar and Scanning active, so you can still see enemies on the map.
If you wipe out, you'll be given an option to challenge the easier optional dungeon Rock Path instead.
A wider variety of items including orbs and Link Boxes can be found on the ground.
